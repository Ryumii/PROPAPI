# ブランド命名の音声学的・心理学的法則リサーチ

> Alpacaの「Aから始まる x パ行を含む」という命名法則を起点に、ブランド名・社名の命名における音声学的・心理学的法則をリサーチした結果。

---

## 1. 「Aから始まる」ことのメリット

### リスト上位表示効果

- **Amazon**: ベゾスは電話帳やサイト一覧でアルファベット順の先頭に来ることを狙い、意図的に「A」始まりの単語を探した（前身の「Cadabra.com」は「cadaver（死体）」と聞き間違えられて変更）
- **APAホテル**: Always Pleasant Amenityの頭文字。ネット検索やリスト表示で上位に来やすく、海外顧客にも覚えやすい戦略

### 母音「ア」の音声特性

- 最も口を大きく開く母音で、「明るい・強い・元気・発展的」な印象を与える
- Oxford大学の研究で、母音始まりのブランド名は人気ブランドにおいて統計的に過剰表現されていることが確認されている

### 現代の注意点

- 検索エンジンはアルファベット順で並べないため、むしろ **K, Q, X, Z** など出現頻度の低い文字の方が独自性で有利という見方もある

## 2. 「パ行（P音 / 破裂音）」の心理的効果

### 破裂音の一般的効果（学術的裏付けあり）

- 力強さ・インパクトを与える
- **語頭の破裂音はブランドの再認・想起・認知に極めて効果的**（Oxford大学研究）
- 破裂音を含むブランド名は **収益性と正の相関** がある
- 消費者はこの音素処理を **無意識・自動的** に行っている

### 日本語の「パ行」の特殊性（川原繁人・慶應大学の研究）

- パ行は江戸時代以前の日本語にほぼ存在せず、外来語で普及した音 → **「洋風・異国的」な響き**
- **かわいらしさ・親しみやすさ**: 子どもが好む音で、お菓子の命名に多用
- **軽快さ・明るさ**: 歯切れの良い破裂が「ポップな印象」を作る

### グリコの「パ行戦略」

ほぼ全商品にパ行が入っている:

- ポッキー（Pocky）
- プリッツ（Pretz）
- パピコ（Papico）
- パナップ（Panapp）
- プッチンプリン（Pucchin Pudding）
- カプリコ（Caplico）
- ペロティ（Peroty）

## 3. 音象徴（Sound Symbolism）の学術的知見

### ブーバ・キキ効果

Ramachandran & Hubbard（2001）の研究で、**被験者の95-98%** が「ブーバ」を丸い形に、「キキ」を尖った形に結びつけた。文化横断的に確認されている。

| タイプ | 音の特徴 | 印象 | 適する商品 |
|---|---|---|---|
| ブーバ型 | 流動子音 m,n,l,r + 後舌母音 | 柔らかい、快適、丸い | 食品、化粧品、癒し系 |
| キキ型 | 硬い子音 k,t,p + 前舌母音 | シャープ、精密、モダン | テック、パフォーマンス系 |

### 前舌母音 vs 後舌母音（Stanford大学 Yorkston & Menon, 2004）

- **前舌母音**（イ、エ）→ 「小さい・軽い・速い・冷たい・女性的」
- **後舌母音**（オ、ウ）→ 「大きい・重い・遅い・温かい・男性的」

### ポケモンでの実証（川原繁人）

- ポケモンは進化すると名前に**濁音が増える**ことが統計的に確認
- 濁音が多い名前のポケモンは体重が重く体が大きい
- 日本語・英語・ポルトガル語・ロシア語話者すべてで確認済み

| **サ行** | さわやか、清潔、涼しい | 発音時に口内温度が下がる |
| **タ行** | しっかり、端正、信頼感 | |
| **ナ行** | なめらか、穏やか、優しい | |

| **濁音（ガ・ザ・ダ・バ行）** | 大きい、重い、強い、高級感 | ガンダム、ゴジラ、ヴィトン |
| **半濁音（パ行）** | 軽快、明るい、かわいい、ポップ | ポッキー、パピコ |

### 語尾の母音別イメージ

| 語尾母音 | イメージ | 例 |
|---|---|---|
| **ア** | 明るい、強い、元気、発展的 | インスパイア、アクア |
| **イ** | 意思が強い、繊細、鋭い、小さい | カムリ、マーチ |
| **ウ** | かわいい、デリケート、なめらか | |
| **エ** | 自然、やさしい、おだやか、楽しい | |
| **オ** | 太い、たくましい、安定、力強い | アルファード、エルグランド |

### 促音「っ」の効果

- リズムにアクセントを生み、テンポ感・歯切れの良さを演出
- 促音の後にはパ行音が来やすい規則がある（例: 「葉っぱ」「一本=いっぽん」）
- ブランド例: ポッキー、プッチンプリン、グッチ、ロッテ

### 濁音の効果

- **男性向け・力強さ**: 有声破裂音（B, D, G）が語頭にある名前は男性向け商品に効果的 → ガンダム、ゴジラ、ドラゴンボール
- **高級感**: ルイ・ヴィトン、ディオール、ハーレーダビッドソンなど高級ブランドには濁音が含まれることが多い
- **「ン」を含むと女性は安心感**: ノンノ、アンアン

| **アンパンマン** | キャラクター | 子どもが最初に覚える名前の代表格 |

- **Porsche, Prada**（/p/始まり）

## 6. 命名フレームワーク・法則

### SMILE & SCRATCH テスト（Alexandra Watkins / Eat My Words）

**良い名前の5条件（SMILE）:**

- **S**uggestive: ブランドの何かを暗示している
- **M**emorable: 馴染みのある連想があり覚えやすい
- **I**magery: 聞いた瞬間にビジュアルが浮かぶ
- **L**egs: テーマとして拡張できる（派生商品など）
- **E**motional: 感情的なつながりを生む

**避けるべき7つの落とし穴（SCRATCH）:**

- **S**pelling Challenged: スペルが難しい
- **C**opycat: 競合と似ている
- **R**estrictive: 拡張性がない
- **A**nnoying: 不快感がある
- **T**ame: 地味で目立たない
- **C**urse of Knowledge: 業界人にしか分からない
- **H**ard to pronounce: 発音しにくい

### ネーミングの8つのヒット法則（CHIBICO）

1. **コンセプト伝達性** — 何者かが一瞬で伝わる
2. **音のリズム感** — 意味より先に音で印象が決まる
3. **視認性と短さ** — 3音節以内で認知・想起率1.5倍
4. **意味の奥行き** — 表面はシンプル、裏に物語性
5. **時代適応性** — 今の消費者感覚に合うトーン
6. **ネガティブ回避** — マイナスの意味・語感を含まない
7. **デジタル強度** — SNS・検索で埋もれない独自性
8. **商標安全性** — 登録可能で法的リスクがない

### K効果

- Oxford大学研究で、「K」音で始まるブランド名が辞書頻度（1.1%）に対し人気ブランドでは**8.63%**と大幅に過剰表現
- 代表例: Kodak、Kellogg's、Kit Kat

### 頭文字効果（Name-Letter Effect）

- ノースウェスタン大学ケロッグ経営大学院の研究
- 人は自分の名前のイニシャルと同じ文字から始まるブランドに無意識に好感を抱く

### 音相理論（黒川伊保子 / 感性リサーチ）

- 発音時の口腔内の物理的感覚（舌の筋肉の硬さ、口腔の高さ、息の流速、鼻腔共鳴度など）を64の属性として数値化
- 例:「S」音（サ行）を発音すると口腔内の表面温度が下がり、「さわやか」「すずやか」という清涼感と結びつく
