### 2.4 Transform 処理の詳細

#### 座標系変換

```python
# JGD2011 (EPSG:6668) → WGS84 (EPSG:4326)
from pyproj import Transformer

transformer = Transformer.from_crs("EPSG:6668", "EPSG:4326", always_xy=True)

def transform_geometry(geom):
    """座標系を JGD2011 から WGS84 に変換"""
    return transform(transformer.transform, geom)
```

#### フォーマット統一

| 入力形式 | 出力形式 | 変換ツール |
|---|---|---|
| GML (国土数値情報) | GeoJSON | GDAL `ogr2ogr` |
| シェープファイル | GeoJSON | GDAL `ogr2ogr` |
| CSV (座標付き) | GeoJSON | Python スクリプト |
| PDF (一部自治体) | 手動 → CSV → GeoJSON | OCR + 手動 |

## 3. データ品質管理

### 3.1 品質チェック項目

| チェック | 内容 | 閾値 | アクション |
|---|---|---|---|
| **件数チェック** | 前回比で大幅な増減がないか | ±20% | 警告アラート |
| **空間整合性** | ポリゴンが有効か（self-intersection なし）| invalid = 0 | 自動修復 → 手動確認 |
| **カバレッジ確認** | 都道府県ごとのデータ有無 | 全47都道府県 | 欠損アラート |
| **座標範囲** | 日本の国土範囲内か | 緯度 20〜46, 経度 122〜154 | 異常値除外 |
| **属性値チェック** | 必須フィールドの null チェック | null率 < 1% | 警告アラート |
| **重複チェック** | 同一ポリゴンの重複 | — | 重複除去 |

## 4. 住所正規化エンジン

### 4.1 正規化フロー

```
入力: "東京都渋谷区渋谷2丁目24-12"
  │
  ├─ Step 1: Unicode正規化 (NFKC)
  │   → "東京都渋谷区渋谷2丁目24-12"
  │
  ├─ Step 2: normalize-japanese-addresses (OSS)
  │   → { pref: "東京都", city: "渋谷区", town: "渋谷二丁目", addr: "24-12" }
  │
  ├─ Step 3: 独自拡張（枝番・号パース）
  │   → { ..., block: "24", lot: "12", building: null }
  │
  ├─ Step 4: 正規化住所生成
  │   → "東京都渋谷区渋谷二丁目24番12号"
  │
  └─ Step 5: address_master でジオコーディング
      → { lat: 35.6595, lng: 139.7004, confidence: 0.97 }
```

### 4.2 対応パターン

| パターン | 入力例 | 正規化結果 |
|---|---|---|
| 漢数字 | 渋谷二丁目二四番 | 渋谷二丁目24番 |
| ハイフン区切り | 渋谷2-24-12 | 渋谷二丁目24番12号 |
| 全角数字 | ２丁目２４番 | 二丁目24番 |
| 大字省略 | 渋谷市渋谷24-12 | 渋谷区渋谷二丁目24番（注：市区の補完も） |
| 方書き | 渋谷2-24-12 〇〇ビル3F | 渋谷二丁目24番12号（方書きは分離） |
| 京都の通り名 | 中京区烏丸通御池上ル | → 特殊処理 |

### 4.3 ジオコーディング精度レベル

| レベル | confidence | 意味 | 精度 |
|---|---|---|---|
| 1 | 0.95〜1.0 | 番地号レベルで特定 | ±10m |
| 2 | 0.80〜0.95 | 街区（番地）レベル | ±50m |
| 3 | 0.60〜0.80 | 町丁目レベル | ±200m |
| 4 | 0.40〜0.60 | 大字レベル | ±500m |
| 5 | < 0.40 | 市区町村レベル | ±2km |

> API レスポンスの `meta.confidence` にこの値を返し、利用者がデータの信頼性を判断できるようにする。

## 5. データ法的根拠

### 5.1 利用可能なライセンス

| データ | ライセンス | 商用利用 | 出典表示 | 備考 |
|---|---|---|---|---|
| 国土数値情報 | 政府標準利用規約 2.0 | ✅ 可 | ✅ 必要 | 「国土数値情報」と明記 |
| アドレスベースレジストリ | CC BY 4.0 | ✅ 可 | ✅ 必要 | デジタル庁 |
| 各自治体データ | 自治体個別 | 要確認 | 多くは必要 | 個別に利用規約を確認 |

### 5.2 出典表示のルール

API レスポンスの各データに `source` フィールドを含め、以下を明記:
- データ提供者名
- データセット名
- 最終更新日

GUI 上では結果ページのフッターに出典一覧を表示。
